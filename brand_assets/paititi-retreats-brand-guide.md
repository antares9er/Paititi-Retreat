# Paititi Retreats — Definitive Brand & Website Direction

## Core Brand Identity

Paititi Retreats is a luxury spiritual retreat brand.

The website must feel:
- sacred
- elegant
- calm
- premium
- feminine
- mystical
- timeless
- transformational

The visual language is inspired by:
- sacred geometry
- lotus symbolism
- celestial spirituality
- luxury wellness retreats
- soft divine femininity

The design direction is NOT modern tech, corporate, minimalist startup, or dark cyber aesthetics.

The entire website should communicate emotional calmness, spiritual depth, and premium retreat energy.

# Definitive Design Direction

The website must follow these visual rules:

- Use large open spacing throughout the layout
- Use warm ivory backgrounds instead of pure white
- Use gold accents sparingly and elegantly
- Use lavender and blush tones as secondary accents
- Use thin decorative dividers and sacred geometric details
- Keep layouts centered and balanced
- Use soft gradients instead of hard color transitions
- Use subtle glow effects around sacred elements
- Use smooth rounded corners
- Use elegant hover transitions
- Use slow, calming animations only

The website must NOT:

- look corporate
- look like a startup SaaS website
- use aggressive modern UI trends
- use harsh black sections
- use neon colors
- use sharp edges
- use heavy shadows
- use cluttered layouts
- use loud typography

The visual experience should feel ceremonial, sacred, premium, and emotionally calming.
